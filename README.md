# 🧮 Calculator Web App

A simple and responsive **Calculator Web Application** built using **HTML, CSS, and JavaScript**.
This project performs basic arithmetic operations with a clean UI and smooth user experience.

🔗 **Live Demo:** https://arjun250.github.io/Calculator1/

---

## 🚀 Features

* Perform basic operations: Addition, Subtraction, Multiplication, Division
* Percentage (%) support
* Clear (Cl) and Delete (DEL) functionality
* Responsive and user-friendly UI
* Error handling for invalid expressions
* Stylish neumorphic design

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the app
* **CSS3** – Styling (Neumorphism UI)
* **JavaScript (Vanilla JS)** – Functionality and logic

## ⚙️ How It Works

* User clicks buttons → values are displayed on the screen
* Expression is built dynamically
* When `=` is pressed:

```js
eval(outputScreen.value)
```

* Result is calculated and displayed instantly
